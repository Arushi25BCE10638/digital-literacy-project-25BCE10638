1.	Project Overview:

This repository serves as a Digital Literacy Portfolio for the CSE0001 course at VIT Bhopal University. 
I have curated these resources to help peers navigate professional online profiles, secure communication, and safe internet practices.

Task: Established professional profiles on industry-standard platforms.

Platforms: GitHub and LinkedIn.
